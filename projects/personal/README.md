# Personal Projects

This folder is for personal and creative tech projects — experiments, side projects, open-source ideas, and anything that doesn't fit the "work demo" category.

**All statuses are valid here.** An `idea` with a good one-liner summary is a legitimate entry.

---

## How to Add a Personal Project

1. Copy [`../_template.md`](../_template.md) to `projects/personal/your-project-slug.md`
2. Fill in the frontmatter (at minimum: `title`, `category: personal`, `status`, `summary`)
3. Write as much or as little as you know — update over time
4. The project will automatically appear in the [Skills Atlas](../../README.md) once you add it to the Mermaid graph

## Examples of What Belongs Here

- Creative tools (music, writing, visual art with AI)
- Games or interactive experiences
- Open-source utilities
- Learning experiments ("I wanted to understand X, so I built Y")
- Personal automation scripts that turned into real projects
- Anything you'd show at a demo night or portfolio review
